About EasyVote

Our Purpose

We created EasyVote for a software engineering project at Oakland University. We set out to design an easy to use website that allows people to view, respond to, and create surveys.

How it Works

There are three main ways to use this website. You can

  • View poll results
  • Respond to polls
  • Create your own polls

You may do all of these without creating an account if you wish. If you do not create an account, you can not keep track of surveys you have responded to, and you also won't be able to edit or delete polls you've already created.
